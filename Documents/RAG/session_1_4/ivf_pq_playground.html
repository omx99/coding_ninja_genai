How does IVF search work?

IVF splits your vector space into nlist clusters (Voronoi cells). At query time, it only searches the nprobe nearest clusters instead of everything — like looking in the right filing cabinet first.

Voronoi cells divide the space. Each cluster has one centroid (★). Your query Q searches the nearest nprobe centroids only.

Product Quantization — compressing vectors with codebooks

PQ splits each vector into sub-vectors and replaces each sub-vector with the nearest centroid ID from a small codebook. 192 bytes → 2 bytes. Use the RGB pixel demo below to feel it.
